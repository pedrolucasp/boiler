# Boiler
A dead-simple ready-to-work skeleton project

Normalize.css as a Reset, also includes a main.sass to begin with

**Colors included on main.sass** 
```
#000000 - for text
#ffffff - for background
#1EAEDB - for links

```

It is just *minimalistic*
